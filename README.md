# Inchworm 🐛

A fast, zero-latency scale converter for Rhino 8. Built for digital fabrication and computational modelling workflows.

## Installation

**Option A: Package Manager (Recommended)**
1. Open Rhino 8 and type `PackageManager`.
2. Search for **Inchworm** and click Install.
3. Restart Rhino.

**Option B: Manual Install**
1. Download the `.rhp` file from the [Releases](../../releases) page.
2. Drag and drop the file directly into your Rhino viewport.

## How to Use

1. Run the `Inchworm` command in Rhino.
2. Set your **Scale Ratio** (e.g., 1 : 20).
3. Enter either the **Real-World** or **Model** length. The other will calculate instantly.
4. Click **Copy & Log** to send the exact result to your clipboard and the Rhino command history.
